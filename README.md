### commtalk - 커뮤니티

### 진행 상황

- 프론트엔드 : ui(scss) 완료 | 백엔드 연결(axios) 미완료
- 백엔드 : 메인화면 진행중

### 사용 기술 스택

- front-end: vue, scss
- back-end: spring boot
- db: jpa, mysql

### 작업 인원

- 김혜란: ui 구성 및 프론트엔드
- 이유리: db관리 및 백엔드 <br/><br/>
